Als Projekt habe Ich ein Bank Management System erstellt .Das Ziel dieses Projekts war die Implementierung von Windows Forms mit C++
und dafür habe Ich nur einfache Funktionen wie Substraktion und Addition implementiert .

Hauptteiele des Projekts :
1-user.h :
Ich habe ein "user" Klasse erstellt mit folgender Attributes : username , password und Balance 

2-Datenbank.h : 
im Datebank.h habe ich ein vector<user> erstellt und dazu habe Ich ein Paar users hinzugefügt.

3-Login :
der Benutzer muss erstmal mithilfe ein username und password einloggen . Daten werden aus dem vector
statt aus einer Datenbank gesucht , um den Prozess zu vereinfachen .

4-Mainform : 
in der Mainform kann der Benutzer einige einfache Aktionen ausführen (deposit/withdraw/hide balance)

Hinweis : als Beispiel Sie können mit folgende Daten einloggen ( username : Rayen / password : example123 )
